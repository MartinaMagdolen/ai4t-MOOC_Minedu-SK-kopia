## Vytvorte si vlastné školenie pomocou GitHub

Na vytvorenie a umiestnenie AI4T Mooc a doplnkových školiacich zdrojov sme sa vo vzdelávacom laboratóriu Inria rozhodli použiť GitHub, nástroj, ktorý bežne používajú vývojári, t. j. **forge**.

Toto riešenie nám umožnilo spoluvytvárať, organizovať, udržiavať, prekladať a zdieľať vzdelávacie zdroje určené pre AI4T. Je tiež open source a zadarmo.

## Ako používať Github na vytvorenie vlastného vzdelávacieho kurzu AI

Táto časť **Vyrobte si vlastné školenie** je metodika, ktorá krok za krokom podporuje tých, ktorí chcú použiť zdroje AI4T Mooc, preložiť ich a usporiadať tak, aby si mohli vytvoriť vlastné "školenie AI pre učiteľov".
